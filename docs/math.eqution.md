# mathjax
add below to html
```js
<script
  src="https://cdn.mathjax.org/mathjax/latest/MathJax.js?config=TeX-AMS-MML_HTMLorMML"
  type="text/javascript">
</script>
```

use \\( your equation \\) and \\[ your equation \\] for inline and display style equations, respectively

quick use
```
\\( \\)
\\[ \\]
```
